 як створити проєкт для управління завданнями та налаштувати його для автоматизації відстеження прогресу.
 як формулювати завдання як Issues, використовувати позначки, встановлювати віхи  та додавати завдання до проєкту.
 як коментувати пропозиції інших членів команди, призначати відповідальних і обговорювати завдання.
 як створювати окремі гілки для виконання завдань, робити коміти та створювати Pull Requests для злиття змін.
 як використовувати ключові слова для автоматичного закриття завдань після злиття Pull Requests.
